
## Height Sense

This is a short, interactive browser-based training game designed to raise awareness on Working at Height (WAH) safety. The game simulates real-life workplace scenarios where users must make decisions to identify hazards, apply proper safety practices, and prevent accidents.

Built as a microlearning tool, the experience takes approximately 3–5 minutes to complete and combines:
- Scenario-based decision making  
- Immediate feedback and learning points  
- Realistic workplace visuals  
- A scoring system to reinforce safe behaviour  

The game is designed for working professionals and can be used as part of workplace safety campaigns, onboarding programmes, or digital LMS modules.
